# Discord Booking Bot

Booking Bot hiện tại chia thành 5 nhóm chính:
- Booking / Bill / Donate.
- Lương / Giờ / Star.
- Shop / Economy.
- Ticket, embed, autoresponder và các công cụ support.
- Cấu hình server, log, anti-raid và payment.

## Ghi chú

- Source hiện tại dùng nhiều command theo vai trò: `owner`, `admin`, `supporter`, `player`, `cash`.
- Một số tính năng chỉ bật trên một số bot name nhất định. Nếu chưa chắc, đánh dấu `TODO: cần xác nhận`.
