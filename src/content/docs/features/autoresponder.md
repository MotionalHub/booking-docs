## Command chính

| Lệnh | Mục đích | Ghi chú |
|---|---|---|
| `ar add` | Thêm responder mới | Prefix |
| `ar edit` | Chỉnh responder | Prefix |
| `ar delete` | Xoá responder | Prefix |
| `ar show` | Xem responder | Prefix |
| `ar list` | Liệt kê responder | Prefix |
| `ar stats` | Xem thống kê | Prefix |
| `ar view` | Xem nội dung responder | Prefix |
